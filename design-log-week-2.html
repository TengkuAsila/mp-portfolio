Web design:

We were introduced to the tools that we will be using to create the website such as GitHub and Atom. We were also introduced to some terms such as HTML, which is the code used to structure a web page and its content and CSS, which is the code use to style the webpage. We then had to pick a template we wanted to use for our website from Bootstrap. Bootstrap is a website that we use to help us create a professional website without starting from scratch. It is like the base of our website which we can then amend to our liking.

By knowing the basic terms as well as the software that we will be using to design the website it will be more helpful for me when I design my website for the assignment.

Photography:

Learned the important elements that needs to be considered before taking pictures such as the type of format we needed to use, the exposure, framing, shutter speed, focus and the image quality.

After knowing the basics of what photography is, it will help me understand it more and what is expected by us when doing the assignments. I will be able to include the basic terms that we were taught in explaining my photograph.
